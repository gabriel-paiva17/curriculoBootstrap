API Presença por Reconhecimento Facial - Link do repositório:

https://github.com/gabriel-paiva17/API-presenca-por-reconhecimento-facial

Projeto pessoal criado com intuito de facilitar o processo de chamada/verificação de presença, esse projeto é uma API RESTful que permite que professores, coordenadores ou gerentes simplifiquem esse processo, utilizando a face das pessoas para que a presença seja validada, ao invés de uma verificação manual. O back-end foi desenvolvido em Go e o banco de dados utilizado foi MongoDB. No frontend, utilizei Vue.js para criar uma interface intuitiva e responsiva. Este projeto ampliou meu conhecimento técnico, e também aprimorou minha capacidade de projetar e implementar soluções robustas e eficientes, além de ter agregado muito para o meu conhecimento de protocolo HTTP.
